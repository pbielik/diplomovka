# Better BibTeX – odporúčaný citekey preset

Odporúčaný formát citekey:

```text
[auth:lower][year][shorttitle3_3]
```

Príklady:
- `koo2016icc`
- `fredrickson2004broaden`
- `levitt2020qualitative`

## Praktické pravidlá
- bez diakritiky
- bez medzier
- malé písmená
- citekey má byť stabilný
- ak je názov príliš dlhý, uprav citekey ručne

## Auto-export
Nastav auto-export celej kolekcie Diplomovka do:
`references/zotero-thesis.bib`

Odporúčanie:
- Better BibTeX JSON nepoužívaj ako hlavný workflow
- pre repozitár stačí `.bib`
