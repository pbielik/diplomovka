# zotero-thesis-seed.bib

Tento súbor je seed šablóna s uzamknutými citekeys pre drafty v markdown.

Odporúčaný workflow:
1. Importuj reálne zdroje do Zotera.
2. Nastav citekeys v Better BibTeX tak, aby presne sedeli s týmto seed súborom.
3. Nechaj Better BibTeX auto-exportovať finálny `zotero-thesis.bib`.
4. V markdown draftoch používaj citekeys z tohto seed súboru.
