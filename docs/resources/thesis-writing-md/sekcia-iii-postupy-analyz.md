# Sekcia III postupy analyz

Aké sú najvýraznejšie prejavy skúmaných premenných? Frekvenčná analýza odpovedí na položky

Aká je vnútorná konzistencia položiek? Cronbachova alfa, McDonaldova omega

Ako môžeme opísať premenné? Priemer, SD Medián, IQR Skw, Kurt, Min, Max, Range

Ako môžeme znázorniť premenné? Boxplot, histogram

Ako silné sú vzťahy medzi premennými? súčinová korelácia, poradová korelácia, korelačný pomer, kontingencia

Aké sú prediktory skúmaných premenných? Regresná analýza (jednoduchá, viacnásobná, binárna)

Ako sú charakterizované empirické typy participantov? K-Means

Aké sú mediátory vzťahu medzi skúmanými premennými? Mediačná analýza

Ako veľké sú rozdiely medzi dvomi/viacerými skupinami? Cohenovo d

Sú rozdiely štatisticky významné? T test, U test, F test, K-W test
