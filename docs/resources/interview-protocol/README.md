# Interview protocol

Tento priečinok obsahuje referenčný schválený interview pack aj finálnu AI-only runtime verziu pre simulované klinické interview s depresívnou symptomatikou.

- `interview_protocol_ai_only_runtime.md` je aktuálny pracovný interview pack pre AI-only runtime použitie.
- `Interview_protocol_AI_only_runtime_final.docx` je finálny dokument exportovaný z aktuálnej AI-only runtime verzie.
- `interview_protocol_ai_only_runtime.txt` je plain-text export AI-only runtime verzie pre skripty, prompty alebo rýchle kopírovanie.
- `interview_protocol_ai_only_runtime_changelog.md` je stručný changelog úprav pri prechode na AI-only runtime framing.
- `Interview_protocol_schvalena_verzia_final_v2.docx` je originálny schválený snapshot.
- `interview_protocol_schvalena_verzia_final_v2.md` je technicky vyčistený prepis schváleného snapshotu do Markdownu bez obsahovej zmeny.

Materiál je uložený v `docs/resources/`, pretože ide o metodický pracovný podklad mimo hlavného rukopisu IMRaD.
